Interview

In deze sectie ga ik vertellen over een interview ik heb gehaad met mij medestudent, Denis.

Waarom heb je deze opleiding gekozen?

IIk heb deze opleiding gekozen, want ik wilde altijd iets doen met computers. Sinds ik klein was, vond Ik vind games heel erg leuk. Dan dacht ik: waarom zou ik niet mijn eigen games maken? Dan na dat ik klaar Was met vmbo. ging ik aanmelden voor deze opleiding. Ik weet dat webdevelopment niet game development is, maar je Moet ook leren hoe je moet programmeren, niet alleen de syntax of de taal, maar hoe je moet denken als een programmeur. En ik denk bij deze opleiding ga ik dat leren.

Heb je ervaring met programmeren?

Ja, ik heb wel een klein beetje ervaring met programmeren. Ik ben begin dit jaar (2024) echt Begonnen met game development te leren. Natuurlijk ben ik nog niet zo ver, want ik heb wel mijzelf Uitgesteld, maar ik zou zeggen dat ik ben tussenliggend met de taal "Lua". Maar als je zou vragen: 'Heb je ervaring met HTML-CSS?' dan zou ik nee zeggen. Dit is mijn eerste keer Dat ik deze twee talen ga leren, en in het begin was het echt moeilijk, want ik moet anders denken dan met Lua.

Hier kan je geen functies maken (ik denk dat je JavaScript moet gebruiken voor dat), geen wiskunde. gebruiken want HTML is meer Meer als een Notepad dan een echte taal. CSS is ook een beetje raar voor mij, want als ik wilde iets Stylen in Lua, zou ik het in dezelfde code kunnen stylen? Maar eigenlijk mag je dat niet doen hier (kan wel, maar het is gewoon beter om een stylesheet te gebruiken).

En dit is een kleine tabel om te laten zien mij Skills als een programmeur

Taal Vaardigheid Tijdstip
HTML Beginner 2 en half een maandjes
CSS Beginner 2 en een half maandjes
Lua Tussenligend 8 maanden

Wat wil je later doen?

Ik wil later een gamedeveloper worden, of misschien applicaties bouwen. Als ik zei op de eerste Vraag: games waren iets dat ik altijd leuk heb gevonden, en zo wil ik ook andere mensen laten genieten van iets ik vind leuk.

Dit betekent dat ik heel veel moet leren van hoe je een game aantrekkelijk moet maken, leren over Rekenen en over natuurkunde gaat wel lukken.

Wat denk je van AI?

Ik ben van mening dat AI gewoon een 'assistent' voor je is. Het helpt je met de makkelijk dingen te Doen maar jij, de persoon moet denken voor de ai. Ik gebruik het meer als een docent dan assistent. Maar ik weet gewoon dat AI iets is dat gewoon blijft voor altijd. En je moet of AI gebruiken, of je gaat gewoon achter blijven.

Het enige nadeel van AI is dat het HEEL makkelijk is om het als een kreuk te gebruiken. Het is Ontzettend makkelijk om AI te vragen: "Maak dit voor mij", maar als het staat op de eisen van deze Periode, je leert gewoon niks als AI alles voor jou doet. Dat is waarom ik het alleen gebruik. Als ik iets niet weet. Het is gewoon mijn persoonlijke docent.

Wat wil je later bereiken?

Ik wil later een echt goede game maken. Ik wil dat de meeste mensen mijn game leuk vinden. Ik wil ook eigenlijk niet meer naar mijn notities kijken (of zo min mogelijk), want ik ken ze uit mijn hoofd. Ik denk dat Ik wil ook gewoon van thuis kunnen werken.
